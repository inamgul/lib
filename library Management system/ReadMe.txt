                                                                  Library Management System

This project is made on c++ language . 

Description Of Project:

This project is built with primary objective to facilitate the school libraries to keep track of there books.In this Library Management System project, you can enter the record of new books and retrieve the details of books available in the library. You can issue the books to the students and maintain their records. Late fine is charged for students who returns the issued books after the due date.Only one book is issued to students.
